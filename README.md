# Luna-2.5B (Antigravity Edition)

An advanced, ultra-lightweight Small Language Model (SLM) engineered to make complex knowledge completely weightless.

## Overview
Built on top of the open-source Qwen2.5-1.5B architecture, Luna-2.5B is optimized for environments with 8GB RAM and i3 CPUs.

## Usage
Refer to the deployment instructions in the main repository for building and deploying using Ollama.
